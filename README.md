# notasAtletas
## É o segundo projetor de certificação do cursor DEVstart
    Essa aplicação criar uma classe que receber uma matriz com os seguintes dados de um atleta o seu nome, idade, peso, altura, e suas notas da competição de ginastica artistica que ele participou. Utilizando esses dados é calculado seu IMC, media válida desconsiderando a maior e menor nota que o atleta recebeu, e sua categoria a qual depende de sua idade. esses dados são calculado e mostrado para o usuario.
